# FuncZ
  FuncZ é uma plataforma desenhada para funcionar como uma ferramenta de geração de cartazes para a empresa "Zenir Móveis", o intuito é ter uma aba onde você tem total controle da busca de produtos (por código), insere valores, altera métodos de pagamento, parcelamentos, etc. A ideia é melhorar os processos internos e mesmo tendo uma ferramenta "oficial" feita pela própria equipe da empresa, nada como ter uma opção que pode fazer um pouco melhor, mudando apenas alguns pontos.

  Dentro do sistema, há várias funcionalidades, algumas mais desenvolvidas que outras, o foco total é a geração dos cartazes, mas há um espaço para o faturamento, com uma maneira inovadora de gerar esse documento pronto e preparado.
