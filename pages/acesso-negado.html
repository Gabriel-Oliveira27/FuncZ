Serviço encerrado

Sistema de Cartazes & Declarações

O acesso ao sistema
foi descontinuado

As ferramentas gratuitas que sustentavam este sistema chegaram ao limite de uso e, sem investimento para substituí-las, não é possível manter o serviço ativo para todos os usuários.

Cota de ferramentas esgotada

Os serviços gratuitos utilizados (APIs, hospedagem e automações) atingiram seus limites e não podem ser renovados sem custo.

Sem investimento disponível

A manutenção de planos pagos não foi viabilizada, inviabilizando a continuidade das funcionalidades para a rede geral.

Obrigado pelo uso

Este sistema foi criado com dedicação. Agradecemos a todos que utilizaram e confiaram na ferramenta durante sua operação.

Acesso ainda ativo para

Usuários da filial de Iguatu, usuários com perfil Admin ou Suporte, e contas específicas cadastradas como exceção ainda conseguem acessar o sistema normalmente. Caso você seja um desses, verifique se realizou login corretamente.

Encerrado

Acesso geral bloqueado. Funcionalidades de cartazes, declarações e relatórios foram suspensas para usuários fora das exceções configuradas.

Último aviso

Limite das ferramentas atingido. APIs externas de geração de documentos e automações via Google Apps Script superaram o volume permitido no plano gratuito.

Em aberto

Retomada depende de investimento. Uma versão paga poderia reativar todos os recursos. Entre em contato com o desenvolvedor caso haja interesse.

Se acredita que deveria ter acesso, entre em contato com o administrador do sistema ou desenvolvedor responsável.
